Bug Bounty

GenCyberCoin website has a bug bounty program! This program will reward you with coins if you find bugs (problems, errors) on the website.

Try to do something that the website would not expect a normal user to do. Afterwards, you may see a message that you found a bug and you will automatically receive a reward for that! For instance, if you submit something on the website, the information will be processed by the server and the result will be shown to you on the page. What if you send something that a normal user would never think to send? Maybe you find a place where you could request a certain type of information from the server or you can try to tinker around on the page itself.

  1. Sensitive data exposure (hidden page that admin left somewhere in one of the pages' source code)
  2. Broken authentication (find admin credentials)
  3. Find a proposed standard which allows websites to define security policies (search this in the Internet and you shall stand on the path to find the answer to this one)
  4. User data input overflow (check out maxlength in page sources)
  5. Reflected XSS (check out wallet)
  6. Stored XSS (btw, did you leave us the feedback yet?)
  7. User input validation (wallet)
  8. HTML edits on social engineering page (how does the server know what question relate to what answer?)
  9. Have you thought what happens if you transfer coins to yourself? (see in the wallet page source what "id" would be yours and then change it)
  10. Local file inclusion (did you see that there are several pages of the items on the market?)
  11. HTML edits on market items (can you change the price of the items so that the money will actually come to you when you buy, instead of going out of your pocket?)
  12. Race condition (when market is enabled, notice what changes in every item's HTML code when it is your turn; then, when it is not your turn, can you still try to buy something?)
  13. Refresh on remove (when market is on)
  14. Access a page that does not exist

P.S.: If you see a page with yellow-ish background and some random code that does not make sense to you, that means you found a real bug that was not supposed to be there. In that case, please email your findings to fordv@arcadia.edu ) with an attached screenshot. Depending on the severity of the real bug, you may be rewarded outside of this camp/class.
