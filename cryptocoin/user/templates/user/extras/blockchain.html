Blockchain

A Blockchain is a public ledger (a keeper of transactions) that is used in cryptocurrencies. The Blockchain allows users to know exactly what transactions happened at any time.

This Blockchain keeps track of all coins transactions and performs the necessary calculations so that users could trust the Blockchain. For instance, every transaction stores a virtual fingerprint (a hash value) of the previous transaction and users can verify that a transaction has truly happened by recalculating that fingerprint.
